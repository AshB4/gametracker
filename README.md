# Final Project - Pathway 3: GameTracker

GameTracker is a React + Vite web app that connects to Supabase (REST API with `fetch`) to list, add, and delete favorite games.

## Tech Stack

- React
- Vite
- React Router DOM
- Supabase REST API (with `fetch`)

## Supabase Setup

Create a table called `games` with this SQL:

```sql
create table public.games (
  id bigint generated by default as identity primary key,
  title text not null,
  genre text not null,
  rating int4 not null check (rating between 1 and 5),
  created_at timestamptz not null default now()
);
```

Set table permissions so your anon key can `select`, `insert`, and `delete`.

## Environment Variables

Update `.env` with your project values:

```env
VITE_SUPABASE_URL=https://YOUR-PROJECT-REF.supabase.co
VITE_SUPABASE_API_KEY=YOUR_SUPABASE_ANON_KEY
```

`.env` is ignored in `.gitignore` and should not be pushed.

## Run Locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Open the local URL shown in your terminal.

## App Routes

- `/` -> My Games page (lists all games, allows delete)
- `/add` -> Add Game page (submits form to Supabase then redirects)

## Deploy to Netlify

1. Push this project to GitHub.
2. In Netlify, create a new site from that GitHub repo.
3. Build command: `npm run build`
4. Publish directory: `dist`
5. Add environment variables in Netlify site settings:
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_API_KEY`
6. Redeploy after setting variables.
